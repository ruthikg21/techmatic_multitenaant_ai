Knowledge Base

Add Source URL

After adding URLs, click Scrape All to fetch content so the AI can use it when answering questions.

How It Works

1. Add URLs — paste any page from your website or docs.

2. Scrape All — the backend fetches each page, strips HTML, and stores the clean text.

3. AI uses the content — every chat message injects the stored text into Claude's context so it can answer accurately.

4. Re-scrape any time to pick up updated content on your website.
