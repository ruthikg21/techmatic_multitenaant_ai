Meta WhatsApp Cloud API Setup Guide
1
Create a Meta Developer App
Go to developers.facebook.com, create a new App (type: Business), and add the WhatsApp product to it.
2
Get Your Phone Number ID
In the Meta Developer Console, go to WhatsApp > API Setup. You'll see your Phone Number ID and a test phone number. Copy the Phone Number ID.
3
Generate a Permanent Access Token
Go to Meta Business Settings > System Users. Create a System User, assign the WhatsApp app with full control, and generate a token with whatsapp_business_messaging and whatsapp_business_management permissions.
4
Configure the Webhook
In Meta Developer Console, go to WhatsApp > Configuration. Set the Callback URL to the webhook URL shown in the Configuration tab. Copy the auto-generated Verify Token from the Configuration tab and paste it as the Verify Token in Meta. Subscribe to the "messages" field.
5
Enter Credentials Here
Paste your Phone Number ID and Access Token in the Configuration tab. The Verify Token is auto-generated for you — just copy it. Enable the integration and save.
6
Test the Connection
Click "Test Connection" to verify your credentials. Then send a WhatsApp message to your business number — the AI bot will reply using your AI Settings configuration.
7
Go Live
Add a real phone number in Meta Business Suite, verify it, and update the Phone Number ID here. Submit your app for App Review with the whatsapp_business_messaging permission to message users beyond the 24-hour window using templates.
